# UTAU Brasil - Índice de Tutoriais

Bem-vindo ao repositório que iniciei de tutoriais UTAU Brasil. Aqui você encontrará todo o conhecimento necessário para começar sua jornada na síntese de voz. Tudo que está posto aqui é público então vocês podem utilizar todos os arquivos daqui e traduzir para outros idiomas, adaptar, editar, distribuir, etc.
